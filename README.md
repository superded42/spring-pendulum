# Симулятор колебаний пружинного маятника
---
В одной программе представлены 2 варианта движения пружинного маятника в горизонтальной плоскости:
- Идеальное движения без сил сопротивления.
- Вариант, приближенный к реальности, с учетом силы сопротивления воздуха, сухого трения и трения покоя.
---
Реальное движение смоделировано численным методом Рунге-Кутты 4-ого порядка.
